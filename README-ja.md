## ビジョン

私たちは、最も強力なAIはすべての人のものであるべきだと考えています。fal'Cieは、妥協なくゼロから訓練された完全なオープンウェイトの大規模言語モデルです。世界中の研究者、開発者、ビルダーが自由に利用できます。APIキー不要。使用制限なし。実行、ファインチューニング、そして所有できます。
